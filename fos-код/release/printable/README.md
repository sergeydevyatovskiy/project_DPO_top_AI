# Печатные формы ФОС

В данной директории размещаются PDF-версии оценочных листов для использования 
в офлайн-режиме (защита перед комиссией, peer-review, оценка руководителем).

## 📄 Состав

| Файл | Назначение |
|------|------------|
| `commission_form.pdf` | Оценочный лист члена комиссии |
| `peer_review_form.pdf` | Шаблон рецензии peer-review |
| `supervisor_form.pdf` | Оценочный лист руководителя КП |
| `student_selfcheck.pdf` | Чек-лист самопроверки студента |
| `commission_conclusion.pdf` | Итоговое заключение комиссии |

## 🖨️ Использование

1. Распечатайте необходимое количество экземпляров;
2. Заполняйте во время защиты / peer-review;
3. Собирайте в портфолио студента после защиты.

## 🔄 Обновление

PDF-формы генерируются из Markdown-шаблонов (`src/templates/*.md`) при каждом 
релизе ФОС. При изменении шаблонов — перегенерируйте PDF.
