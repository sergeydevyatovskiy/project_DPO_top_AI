## 4. Ролевые «портреты» уровней освоения

### ML Researcher (Исследователь МО)

| Балл | Уровень | Характеристика |
|------|---------|----------------|
| **4** | Исполнитель | Применяет готовые алгоритмы оптимизации по инструкции, использует стандартные методы настройки гиперпараметров (поиск по сетке). |
| **6** | Самостоятельный | Самостоятельно выбирает метод оптимизации и обосновывает его применение; настраивает гиперпараметры с использованием случайного поиска и базовой байесовской оптимизации. |
| **8** | Проектировщик | Сравнивает методы оптимизации, интерпретирует результаты экспериментов, аргументирует выводы; адаптирует классические методы МО под специфику задачи. |
| **10** | Архитектор-визионер | Предлагает модификации существующих методов оптимизации; разрабатывает комплексные стратегии обучения моделей с учётом требований к производительности и масштабируемости. |

### AI Security Engineer (Инженер безопасности ИИ)

| Балл | Уровень | Характеристика |
|------|---------|----------------|
| **4** | Исполнитель | Знает базовые принципы устойчивости моделей, применяет стандартные техники борьбы с переобучением по инструкции. |
| **6** | Самостоятельный | Обосновывает выбор методов повышения устойчивости и безопасности; применяет базовые методы предобработки данных с учётом устойчивости моделей. |
| **8** | Проектировщик | Оценивает результативность методов устойчивости на основе сопоставления с аналогами; анализирует влияние методов оптимизации на надёжность моделей. |
| **10** | Архитектор-визионер | Разрабатывает комплексные стратегии повышения устойчивости, надёжности и безопасности моделей; формирует рекомендации по улучшению с учётом современных методологий. |
