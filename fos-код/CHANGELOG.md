# Changelog

Все существенные изменения в ФОС фиксируются в этом файле.  
Формат основан на [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] — 2026-07-14

### Добавлено
- Полный комплект артефактов ФОС в соответствии с ТЗ;
- Паспорт ФОС (`docs/passport.md`);
- Матрица соответствия КРМ × оценочные средства (`docs/matrix.md`);
- Критерии оценивания курсового проекта по двум осям;
- Регламент peer-review и критерии оценки защиты;
- 10-балльные рубрики для обеих осей и защиты;
- Шаблоны оценочных листов (руководитель, комиссия, peer-review, самопроверка);
- Машиночитаемая матрица КРМ (`data/competency_map/mapping.csv`);
- Отчёт пилотной валидации на 3 иллюстративных работах;
- Инструкция для преподавателя-пользователя (`release/how_to_apply.md`);
- ADR по выбору ролей и весов оценочных средств.

### Проверено
- Покрытие всех 8 индикаторов КРМ оценочными средствами (триангуляция);
- Сумма весов = 100% на всех уровнях иерархии;
- Согласованность формулировок между разделами;
- Пилотный прогон ФОС на 3 работах (сильная / средняя / слабая).

### Соответствие
- ПР-1 — ПР-8 ТЗ на разработку ФОС — выполнено.
